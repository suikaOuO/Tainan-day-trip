上次瀏覽的行程：
⚠️ 尚未連接資料庫，目前僅能在這台裝置上使用
要讓朋友輸入代碼後即時看到同一份行程（打卡進度、花費都同步），需要一個免費的 Firebase 專案來存放共用資料：
  1. 到 firebase.google.com 建立一個新專案（免費方案即可）。
  2. 在專案中新增一個「Web app」，把它給你的 firebaseConfig 設定值複製下來。
  3. 啟用左側選單的 Firestore Database，選擇「以測試模式啟動」建立資料庫。
  4. 打開這個檔案，找到最上方 FIREBASE_CONFIG 的地方，貼上你的設定值。
  5. 建議到 Firestore 的「規則」分頁，把規則改成只允許 trips 這個集合被讀寫，例如：
    match /trips/{code} { allow read, write: if true; } （因為沒有登入機制，知道代碼的人都能編輯，請不要把代碼公開分享給不信任的人）。
  6. 存檔後重新上傳／部署到 GitHub Pages，就可以真正跟朋友共用行程了。
